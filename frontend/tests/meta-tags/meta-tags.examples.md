# Тестирование метатегов

Этот документ описывает тесты для проверки метатегов на всех страницах сайта.

## Что проверяется

### Обязательные метатеги

- **title** - заголовок страницы (должен быть не пустым, минимум 10 символов)
- **description** - описание страницы (должен быть не пустым, минимум 10 символов, максимум 160 для SEO)

### Open Graph метатеги

- **og:title** - заголовок для социальных сетей
- **og:description** - описание для социальных сетей
- **og:image** - изображение для социальных сетей
- **og:type** - тип контента (минимум 2 символа)
- **og:locale** - локаль (минимум 2 символа)

### Twitter Card метатеги

- **twitter:card** - тип карточки Twitter (минимум 2 символа)
- **twitter:title** - заголовок для Twitter
- **twitter:description** - описание для Twitter
- **twitter:image** - изображение для Twitter

### Дополнительные метатеги (если есть)

- **robots** - инструкции для поисковых роботов
- **canonical** - канонический URL

## Запуск тестов

```bash
# Запустить только тесты метатегов
npm run test:meta-tags

# Запустить все общие тесты
npm run test:general
```

## Примеры корректных метатегов

### HTML разметка

```html
<head>
    <title>Главная — Название сайта</title>
    <meta name="description" content="Описание страницы для поисковых систем" />
    <meta name="viewport" content="width=device-width, initial-scale=1" />

    <!-- Open Graph -->
    <meta property="og:title" content="Название сайта — Главная" />
    <meta property="og:description" content="Описание страницы для социальных сетей" />
    <meta property="og:image" content="/build/img/og-image.jpg" />
    <meta property="og:type" content="website" />
    <meta property="og:locale" content="ru_RU" />

    <!-- Twitter Cards -->
    <meta name="twitter:card" content="summary" />
    <meta name="twitter:title" content="Название сайта — Главная" />
    <meta name="twitter:description" content="Описание страницы для Twitter" />
    <meta name="twitter:image" content="/build/img/og-image.jpg" />
</head>
```

### Данные страницы (front matter)

```yaml
---
title: Главная
description: Описание главной страницы сайта
og_image: /img/custom-og-image.jpg
---
```

## Ошибки и их решения

### Ошибка: "Метатег должен быть не пустым"

**Причина:** Метатег отсутствует или имеет пустое значение
**Решение:** Добавить метатег с корректным значением

### Ошибка: "Метатег должен содержать минимум X символов"

**Причина:** Метатег слишком короткий
**Решение:** Увеличить длину значения метатега

### Ошибка: "Description не должен превышать 160 символов"

**Причина:** Описание слишком длинное для SEO
**Решение:** Сократить описание до 160 символов или меньше

## Настройка тестов

Тесты автоматически находят все страницы в `src/pages/` и проверяют их метатеги. Для добавления новых страниц просто создайте HTML файл в этой директории.

## Логирование

При запуске тестов в консоль выводится информация о найденных метатегах для каждой страницы, что помогает в отладке.
